# Mini Todo



一个简单的待办事项微信小程序，具有以下功能：
- 添加新的待办事项
- 标记待办事项为已完成
- 删除待办事项
- 查看所有/已完成/未完成的待办事项
- 数据本地存储

## 技术特点
- 使用微信小程序原生框架开发
- 使用 wx.setStorageSync 实现本地数据持久化
- 简洁直观的用户界面
